# Smali Support

<img src="assets/smali-icon.png" width="64"/>

A smali/baksmali plugin for [Xed-Editor](https://github.com/Xed-Editor/Xed-Editor).

## What it does
- Syntax highlighting for `.smali` files
- Assemble `.smali` → `classes.dex`
- Disassemble `.apk` / `.dex` → `.smali`

## Install
Download `SmaliSupport.zip` from [Releases](../../releases) and install via **Xed → Extensions → Install from storage**.

## Requirements
```bash
pkg install openjdk-17 curl
```
